## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/zustand_react_query.git
```

Go to the project directory

```bash
  cd zustand_react_query
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
